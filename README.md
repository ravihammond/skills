# Skills

[![skills.sh](https://skills.sh/b/ravihammond/skills)](https://skills.sh/ravihammond/skills)

Public Codex skill collection from `ravihammond/skills`.

## Install with npx skills

List the available skills:

```bash
npx skills add ravihammond/skills --list
```

Install all skills globally for Codex:

```bash
npx skills add ravihammond/skills --skill '*' --agent codex --global --yes
```

Install one skill:

```bash
npx skills add ravihammond/skills --skill coding-research --agent codex --global --yes
```

Available skills:

- `coding-research`
- `decision-handoff`
- `implement-issue`
- `review-issue`
- `tdd`

## Install with gh skill

List the available skills:

```bash
gh skill install ravihammond/skills
```

Install all skills globally for Codex:

```bash
gh skill install ravihammond/skills --all --agent codex --scope user
```

Install one skill:

```bash
gh skill install ravihammond/skills coding-research --agent codex --scope user
```
